Installation
============

Use the in-game Add-on Modules. Search for 'item piles'.

Manual Installation
-------------------

1. As above, use the 'Add-on Modules' dialog.At the bottom of this dialog you will find the :guilabel:`Manyfest URL` field.
2. In the :guilabel:`Manifest URL` enter the following: ``https://github.com/fantasycalendar/FoundryVTT-ItemPiles/releases/latest/download/module.json``.
3. |LMB| 'Install' and wiat for the installation to complete.
4. In your Game  World remember to activate the Item Piles module.
